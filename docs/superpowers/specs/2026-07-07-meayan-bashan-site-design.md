## דרישת ליבה: עריכה עצמאית

מעיין תעדכן את התוכן בעצמה (הרצאות קרובות, אזכורים תקשורתיים וכו') **בלי לגעת בקוד**, אחרי שהאתר עולה לאוויר.

## ארכיטקטורה

- **Astro** — מחולל אתר סטטי, פלט HTML/CSS מהיר, תמיכה מלאה ב-RTL בעברית.
- **Decap CMS** (git-based CMS) — פאנל ניהול תוכן בכתובת `/admin`, אימות דרך Netlify Identity. כל שמירה של מעיין יוצרת commit בריפו ומפעילה דיפלוי אוטומטי.
- **Netlify** — אחסון + build + forms, tier חינמי. עלות בפועל: רק חידוש הדומיין הקיים (כבר בבעלות הלקוחה, יחובר בסוף התהליך דרך DNS).
- **Netlify Forms** — טפסי יצירת קשר (הכללי והעסקי) ללא צורך בשרת/בסיס נתונים; הגשות מגיעות למייל של מעיין.

**נשקלו וסוננו:** וורדפרס (עלות אחסון חודשית + נטל תחזוקת אבטחה), Flask מותאם אישית + פאנל ניהול על Render (התאמה לתשתית הקיימת של הלקוח אבל עבודת פיתוח מיותרת לאתר תדמית לא-מורכב).

## שפה

עברית בלבד, RTL. אין תמיכה רב-לשונית ב-MVP.

## מפת אתר

1. **בית** — הירו (תמונה + משפט פתיחה) → סרגל "כפי שהופיעה ב" (לוגואים) → טעימה מהאודות → טעימה מהרצאה קרובה → סעיף המלצות → CTA ליצירת קשר
2. **אודות** — סיפור אישי ורקע מקצועי (טקסט חופשי, ניתן לעריכה מלאה ב-CMS)
3. **תקשורת** — רשימת אזכורים, מסוננת לפי קטגוריה: טלוויזיה / פודקאסטים / כתבות ופינות / אחר. כל פריט: כותרת, קטגוריה, תאריך, קישור חיצוני, תמונה ממוזערת
4. **הרצאות וקורסים** — רשימת הרצאות/קורסים: כותרת, תיאור, פורמט. **מידע בלבד** — אין הרשמה/תשלום מובנה באתר, רק כפתור/קישור ליצירת קשר חיצוני (טלפון/וואטסאפ/מייל)
5. **ארגונים** — תיאור השירותים לחברות (הרצאות, סדנאות וכו') + **טופס יצירת קשר עסקי נפרד** מהטופס הכללי
6. **המלצות** — מוצג כסעיף בדף הבית; ניתן להטמיע גם בדפים רלוונטיים אחרים (למשל הרצאות וקורסים)
7. **צור קשר** — טופס כללי + קישורים לרשתות חברתיות

ניווט עליון (RTL): בית · אודות · תקשורת · הרצאות וקורסים · ארגונים · המלצות · צור קשר

## מודל תוכן (Decap CMS)

- **settings** (קובץ יחיד): כותרת אתר, טאגליין, תמונת הירו, ביו קצר, טלפון, מייל, קישורי סושיאל (אינסטגרם, טיקטוק, פייסבוק)
- **about** (קובץ יחיד): טקסט ביו מלא (rich text)
- **media** (אוסף): כותרת, קטגוריה (tv / podcast / article / other), תאריך, קישור חיצוני, תמונה, תיאור קצר
- **courses** (אוסף): כותרת, תיאור, פורמט, קישור/טלפון ליצירת קשר חיצוני, תמונה
- **organizations** (קובץ יחיד): תיאור השירותים, רשימת נקודות
- **testimonials** (אוסף): שם הממליץ/ה, ציטוט, תמונה/מקור (אופציונלי)
- **partner_logos** (אוסף): רשימת לוגואים לסרגל "כפי שהופיעה ב" — שם הגוף + קובץ לוגו
